Keys in R3R
Key Description Notes
Key Description Notes
d Donate
g Go to a URL
h This help Same as ?
j See the next item Goes to the first item if on the last item.
k See the previous element Goes to the last item if on the first item.
Ctrl+l Refresh the view
o Options menu
q Exit the program It's recommended that you push q instead of closing the window, so that used memory will be freed.
? This help Same as h
/ Go to specific item
~ Go between the first and last element This is due to buggy behavior in FPC.
! Start a shell Type exit and press enter to return to R3R. Shell support is buggy in FPC and non-Unix platforms.
<Spacebar> Scroll down
<Backspace> Scroll up
<Enter> Open the item's main link in your browser. You must set your browser in the options first.
